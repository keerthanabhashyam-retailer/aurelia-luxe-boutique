If the AI Studio "Sync to GitHub" button is looping, use this manual method:

   - Drag and drop your project files into the browser.
   - Click "Commit changes".
5. **Vercel**:
   - Go to [Vercel](https://vercel.com) and click "Add New > Project".
   - Import your new GitHub repository.
   - **Environment Variables**: Add a variable named `API_KEY` and paste your Gemini key.
   - Click **Deploy**.
